# Looking for an a

Write a program that reads a sequence of characters ended in a dot and
tells if the sequence has any lowercase '`a`' or not.

## Input

Input consists of a sequence of characters ended in '`.`'.

## Output

Print "`yes`" or "`no`" depending on whether '`a`' appears in the
sequence or not.
