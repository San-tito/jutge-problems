# Factorial iteratiu

Escriviu una funció iterativa que, donat un naturak $n$, retorni el seu
factorial \@n@$!$.

## Interfície

+-------:+:------------------------------------+
| C++    | ``` c++                             |
|        | int factorial(int n);               |
|        | ```                                 |
+--------+-------------------------------------+
| C      | ``` c                               |
|        | int factorial(int n);               |
|        | ```                                 |
+--------+-------------------------------------+
| Java   | ``` java                            |
|        | public static int factorial(int n); |
|        | ```                                 |
+--------+-------------------------------------+
| Python | ``` python                          |
|        | factorial(n)  # returns int         |
|        | ```                                 |
+--------+-------------------------------------+
|        | ``` python                          |
|        | factorial(n: int) -> int            |
|        | ```                                 |
+--------+-------------------------------------+

## Precondició

Es compleix $0\le$ \@n@ $\le 12$.

## Observació

Només cal enviar el procediment demanat; el programa principal serà
ignorat.
