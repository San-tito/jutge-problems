# Racionals (1)

Usant la definició

        struct Racional {
            int num, den;
        };

feu una funció

        Racional racional(int n, int d);

que retorni un racional equivalent a $\frac{n}{d}$ i "normalitzat"
segons la convenció següent: si el numerador és 0, el denominador és 1;
altrament, el numerador i el denominador no tenen cap factor comú, i el
denominador és positiu.

## Precondició

\@d@ $\neq 0$.

## Observació

Només cal enviar el procediment demanat; el programa principal serà
ignorat.

Seguiu estrictament la definició dels tipus de l'enunciat.
