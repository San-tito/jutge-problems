# Recursive factorial

Write a function that returns \@n@$!$.

Solve this problem recursively.

## Interface

+-------:+:------------------------------------+
| C++    | ``` c++                             |
|        | int factorial(int n);               |
|        | ```                                 |
+--------+-------------------------------------+
| C      | ``` c                               |
|        | int factorial(int n);               |
|        | ```                                 |
+--------+-------------------------------------+
| Java   | ``` java                            |
|        | public static int factorial(int n); |
|        | ```                                 |
+--------+-------------------------------------+
| Python | ``` python                          |
|        | factorial(n)  # returns int         |
|        | ```                                 |
+--------+-------------------------------------+
|        | ``` python                          |
|        | factorial(n: int) -> int            |
|        | ```                                 |
+--------+-------------------------------------+

## Precondition

Assume $0\le$ \@n@ $\le 12$.

## Observation

You only need to submit the required procedure; your main program will
be ignored.
