# Rectangles (2)

Write a program that prints $n\times m$ rectangles. Print each rectangle
in an independent way, fill it up with 9, followed by 8, ...followed by
0, etc.

## Input

The input consists of several pairs of natural numbers.

## Output

For every pair $(n,m)$, print a rectangle of size $n \times m$ filled
with 9,8,...0, 9, .... Separate two rectangles with an empty line.
