# Màxim de quatre enters

Feu una funció que, donats quatre enters \@a@, \@b@, \@c@ i \@d@, en
retorni el màxim.

## Interfície

+-------:+:----------------------------------------------------+
| C++,C  | ``` c++                                             |
|        | int max4(int a, int b, int c, int d);               |
|        | ```                                                 |
+--------+-----------------------------------------------------+
| Java   | ``` java                                            |
|        | public static int max4(int a, int b, int c, int d); |
|        | ```                                                 |
+--------+-----------------------------------------------------+
| Python | ``` python                                          |
|        | max4(a, b, c, d)  # returns int                     |
|        | ```                                                 |
+--------+-----------------------------------------------------+
|        | ``` python                                          |
|        | max4(a: int, b: int, c: int, d: int) -> int         |
|        | ```                                                 |
+--------+-----------------------------------------------------+

## Observació

Només cal enviar el procediment demanat; el programa principal serà
ignorat.

## Pista

Usar la funció \@max2(a, b)@ de l'exercici 
`problem://problemsjutge.org:problems/p1/jpetit/maxim-funcio.pbm` podria
ser útil.
