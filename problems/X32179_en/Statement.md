# Related even positions

Write a program which, given a list of integers with at least two
elements, says how many elements in even positions are equal to the
second integer in the sequence. The first integer in the sequence is
considered to be in position 1, the second integer in position 2, etc.

Note that the second element of the sequence must be included in the
count.

**Exam score:** 3.5 **Automatic part:** 50%

## Input

The input consists of a sequence of at least two integers.

## Output

You have to write how many elements satisfy the condition described in
the problem statement.
