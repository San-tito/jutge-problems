# Nombres creixents

En aquest exercici, direm que un natural és creixent si cada dígit és
més petit o igual que el que està a la seva dreta (si aquest existeix).

Feu una funció recursiva que indiqui si un natural \@n@ és creixent o
no.

## Interfície

+-------:+:-----------------------------------------+
| C++    | ``` c++                                  |
|        | bool es_creixent(int n);                 |
|        | ```                                      |
+--------+------------------------------------------+
| C      | ``` c                                    |
|        | int es_creixent(int n);                  |
|        | ```                                      |
+--------+------------------------------------------+
| Java   | ``` java                                 |
|        | public static boolean esCreixent(int n); |
|        | ```                                      |
+--------+------------------------------------------+
| Python | ``` python                               |
|        | es_creixent(n)  # returns bool           |
|        | ```                                      |
+--------+------------------------------------------+
|        | ``` python                               |
|        | es_creixent(n: int) -> bool              |
|        | ```                                      |
+--------+------------------------------------------+

## Observació

Només cal enviar el procediment demanat; el programa principal serà
ignorat.
