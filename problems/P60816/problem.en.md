# Reversed number in hexadecimal

Write a program that reads a number and prints its hexadecimal
representation reversed. Follow the convention to use the letters from
'`A`' to '`F`' to represent the values from 10 to 15.

## Input

Input consists of a natural number.

## Output

Print, reversed, the hexadecimal representation of the number, with as
many zeros at its left as required.
