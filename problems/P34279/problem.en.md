# Add one second

Write a program that adds one second to a clock time, given its hours,
minutes and seconds.

## Input

Input consists of three natural numbers $h$, $m$ and $s$ that represent
a clock time, that is, such that $h<24$, $m<60$ and $s<60$.

## Output

Print the new clock time defined by $h$, $m$ and $s$ plus one second in
the format "`HH:MM:SS`".
