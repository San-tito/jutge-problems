# Ordenació per selecció

Feu un procediment que ordeni un vector de petit a gran utilitzant
l'algorisme d'ordenació per fusió.

## Interfície

+-----:+:------------------------------------------+
| C++  | ``` c++                                   |
|      | void ordena_per_fusio(vector<double>& V); |
|      | ```                                       |
+------+-------------------------------------------+
| MyPy | ``` python                                |
|      | ordena_per_fusio(v: list[float]) -> None  |
|      | ```                                       |
+------+-------------------------------------------+

## Observació

Només cal enviar el procediment demanat; el programa principal serà
ignorat.
