# Rectangles (3)

Write a program that prints $k$, $n\times m$ rectangles. Print each
rectangle in an independent way, fill it up with 9, followed by 8,
...followed by 0, etc.

## Input

The input consists of three natural numbers $k$, $n$, $m$.

## Output

Print $k$ rectangles of size $n \times m$, filled with 9,8,...0, 9, ....
Separate two rectangles with an empty line.
