# Harmonic numbers (1)

Write a program that reads a number $n$ and prints the $n$-th harmonic
number, defined as $H_n = 1/1 + 1/2 + \cdots + 1/n$.

## Input

Input consists of a natural number $n$.

## Output

Print $H_n$ with 4 digits after the decimal point.
