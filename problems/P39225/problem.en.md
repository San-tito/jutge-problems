# I-th (1)

Write a program that, given an integer number $i$ and a sequence of
natural numbers $x_1, \dots, x_n$, prints $x_i$.

## Input

Input begins with a number $i$, followed by the sequence
$x_1, \dots, x_n$ ended with $-1$. It is known that $1 \le i \le n$.

## Output

Print the content of the position $i$ as it is shown in the examples.
