# Function for leap years

Write a function that tells if the given year is a leap year or not. To
recall the rules about leap years, check the exercise 
`problem://problemsjutge.org:problems/p1/jpetit/traspas-elemental.pbm` .

## Interface

+-------:+:--------------------------------------------+
| C++    | ``` c++                                     |
|        | bool is_leap_year(int year);                |
|        | ```                                         |
+--------+---------------------------------------------+
| C      | ``` c                                       |
|        | int is_leap_year(int year);                 |
|        | ```                                         |
+--------+---------------------------------------------+
| Java   | ``` java                                    |
|        | public static boolean isLeapYear(int year); |
|        | ```                                         |
+--------+---------------------------------------------+
| Python | ``` python                                  |
|        | is_leap_year(year)  # returns bool          |
|        | ```                                         |
+--------+---------------------------------------------+
|        | ``` python                                  |
|        | is_leap_year(year: int) -> bool             |
|        | ```                                         |
+--------+---------------------------------------------+

## Precondition

The parameter \@year@ is between 1800 and 9999, both included.

## Observation

You only need to submit the required procedure; your main program will
be ignored.
