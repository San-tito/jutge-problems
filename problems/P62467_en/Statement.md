# Bars (1)

Write a program that reads a natural number $n$, and prints $2^n - 1$
bars of asterisks, following the pattern that can be deduced from the
examples.

## Input

Input consists of a natural number $n > 0$.

## Output

Print $2^n - 1$ lines, following the pattern that can be deduced from
the examples.
