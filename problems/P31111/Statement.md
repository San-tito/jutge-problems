# Parentheses

Given a sequence made up of only '`(`' and '`)`', tell if the
parentheses close correctly.

## Input

Input consists of a sequence of parentheses.

## Output

Print "`yes`" or "`no`", depending on whether the parentheses close
correctly or not.
