# Time decomposition (1)

Write a program that, given a number of seconds $n$, prints the number
of hours, minutes and seconds represented by $n$.

## Input

Input consists of a natural number $n$.

## Output

Print three natural numbers $h$, $m$, $s$ such that $3600h+60m+s=n$,
with $m<60$ and $s<60$.
