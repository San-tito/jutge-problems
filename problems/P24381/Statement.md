# Print a cross

Write a procedure

        void cross(int n, char c);

to print an $n \times n$ cross with the character \@c@. For instance,
the call

        cross(5, 'X');

produces this output:

      X
      X
    XXXXX
      X
      X

## Precondition

\@n@ $\ge 3$, \@n@ is odd.

## Observation

You only need to submit the required procedure; your main program will
be ignored.
