# I-th (3)

Write a program that, given an integer number $i$ and a sequence of
natural numbers $x_1, \dots, x_n$, prints $x_i$.

## Input

Input begins with an integer number $i$, followed by the sequence
$x_1, \dots, x_n$.

## Output

If the position $i$ is correct, print the content of $i$ as it is shown
in the examples. Otherwise, print "`Incorrect position.`".
