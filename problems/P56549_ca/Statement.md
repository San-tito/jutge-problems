# Canvis de base

Useu recursivitat per fer un programa que, per a cada nombre donat,
l'escrigui en notació binària, octal i hexadecimal.

## Entrada

L'entrada consisteix en diversos nombres naturals.

## Sortida

Per a cada nombre de l'entrada, cal escriure la seva representació
binària, octal i hexadecimal seguint el format de l'exemple.
