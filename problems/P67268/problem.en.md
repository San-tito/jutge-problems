# Reverse of sequences

Write a program that reads sequences of integer numbers, and prints each
one reversing the order of its elements.

## Input

Input consists of several sequences. Each sequence consists of a natural
number $n$, followed by $n$ integer numbers.

## Output

Print each sequence in reverse order in a line.
