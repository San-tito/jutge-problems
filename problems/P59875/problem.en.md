# Top to bottom

Write a program that reads two numbers $x$ and $y$, and prints all
numbers between $x$ and $y$ (or between $y$ and $x$), in decreasing
order.

## Input

Input consists of two integer numbers $x$ and $y$.

## Output

Print all integer numbers between $x$ and $y$ (or between $y$ and $x$),
in decreasing order.
