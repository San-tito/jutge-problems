# How many different?

Write a program that prints how many different numbers there are in a
sequence with $n$ integer numbers.

## Input

Input contains different test data. Each test data starts with a natural
number $n\ge1$ followed by a sequence with $n$ integer numbers.

## Output

For each sequence of the input print, in a line, the number of different
elements that contains.
