# Control C202D

Feu un programa que, per a cada parell d'enters $a$ i $b$ donats,
calculi i escrigui la suma de tots els cubs entre $a$ i $b$.

## Entrada

L'entrada és una seqüència de parells d'enters $a$ i $b$.

## Sortida

Per a cada parell de l'entrada, cal escriure, en una línia i segons el
format de l'exemple, el resultat de la suma
$a^3 + (a + 1)^3 + \cdots + (b - 1)^3 + b^3$.
