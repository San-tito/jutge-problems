# Escriviu una creu

Escriviu un procediment

        void cross(int n, char c);

que escrigui una creu $n \times n$ amb el caràcter \@c@. Per exemple, la
crida

        cross(5, 'X');

produeixen aquesta sortida:

      X
      X
    XXXXX
      X
      X

## Precondició

\@n@ $\ge 3$, \@n@ és senar.

## Observació

Només cal enviar el procediment demanat; el programa principal serà
ignorat.
