# Comparant fraccions

Implementeu la funció

        bool c_frac(int n1, int d1, int n2, int d2);

que retorni cert si la fracció $\frac{n1}{d1}$ és estrictament menor que
la fracció $\frac{n2}{d2}$.

## Precondició

\@d1@, \@d2@ $> 0$.

## Observació

Només cal enviar el procediment demanat; el programa principal serà
ignorat.
