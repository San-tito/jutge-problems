# Comparant fraccions

Implementad la función

        bool c_frac(int n1, int d1, int n2, int d2);

que devuelbe cierto si la fracción $\frac{n1}{d1}$ es estrictamente
menor que la fracción $\frac{n2}{d2}$.

## Precondición

\@d1@, \@d2@ $> 0$.

## Observación

Sólo tenéis que enviar el procedimiento requerido; el programa principal
será ignorado.
